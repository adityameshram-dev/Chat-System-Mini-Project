# Chat-System-Mini-Project
using java GUI + Socket Programming 


How to Run: 

**Step 1: Start the Server**
1. Open the file 'Serverchatsystem.java'.
2. Compile and run it:
    javac Serverchatsystem.java
    java Serverchatsystem
3. The server will start on port 5001 and display your local IP.

**Step 2: Start the Client**
1. Open the file 'UI.java' on the same or a different device.
2. Compile and run it:
    javac UI.java
    java UI
3. GUI window will open for the client.

**Step 3: Connect to Server**
1. Enter the Server IP (shown by the server program).
2. Enter your Name.
3. Port: 5001 (by default).
4. Click on Connect button.

Now Start Chatting...

Note:
1. Make sure the Server is running first before starting the Client.
2. Client and Server must be on the same network.
